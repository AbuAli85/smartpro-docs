## The Key: submissionId

The `submissionId` is the **unique identifier** that connects everything:

1. **Created** on marketing site when consultation is submitted
2. **Passed** to platform via URL parameters
3. **Used** to track lead progression via API
4. **Stored** in database to link consultation to user
5. **Displayed** in platform dashboard to show consultation status

**Without submissionId, there's no connection!**
